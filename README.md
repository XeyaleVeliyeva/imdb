# imdb
IMDb Movie Analysis App (2006–2016)

This project is an interactive data analysis web application built with Streamlit. It explores IMDb movie data from 2006 to 2016 and provides insights through dynamic visualizations and filtering options.

Features
Genre-based filtering
Rating analysis
Year-wise movie trends
Search and exploration functionality

How to Run
Clone the repository:
`git clone https://github.com/XeyaleVeliyeva/imdb.git`

Navigate to the project folder:
`cd imdb`

Install dependencies:
`pip install -r requirements.txt`

Run the application:
`streamlit run app4.py`
